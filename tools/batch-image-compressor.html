ℹ️ About This Tool

Compress and resize a whole batch of images in one go — set a maximum width and JPEG quality, then download every processed image packaged in a single zip file.

How to use:

  1. Select multiple images at once.
  2. Set a max width (to resize down large photos) and JPEG quality (to reduce file size).
  3. Click Compress & Download All to get a zip with every processed image.

ℹ️ इस टूल के बारे में

कई इमेज को एक साथ compress और resize करें — max width और JPEG quality सेट करें, फिर हर प्रोसेस की हुई इमेज को एक zip फाइल में डाउनलोड करें।

इस्तेमाल कैसे करें:

  1. एक साथ कई इमेज चुनें।
  2. Max width (बड़ी फोटो को छोटा करने के लिए) और JPEG quality (फाइल साइज़ कम करने के लिए) सेट करें।
  3. Compress & Download All दबाकर सभी प्रोसेस की हुई इमेज वाला zip पाएं।
