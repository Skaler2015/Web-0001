ℹ️ About This Tool

The ApneSoftware QR & Barcode Scanner is a professional, privacy-first tool for decoding QR codes and barcodes directly in your browser — from an image, a screenshot, a clipboard paste or your device's camera. We meet codes everywhere now: a QR on a restaurant table, a WiFi code taped to a wall, a payment code in a chat, an EAN barcode on a product, a Code 128 label on a parcel, or a boarding pass rendered as a PDF417. Often the code arrives as a picture rather than something you can hold up to a phone — a friend forwards a screenshot, a document contains an embedded code, or you need to read a barcode from a photo. This tool solves exactly that: give it a picture or point the camera, and it instantly reads what the code contains and shows you what to do next, all without uploading anything.

At its core is a robust multi-format decoding engine. It reads standard QR codes — including tricky ones that are inverted (light on dark), slightly rotated, or small in the frame — and a wide range of 1D and 2D barcodes: Code 128, Code 39, EAN-13, EAN-8, UPC-A/E, ITF (interleaved 2 of 5), and the dense 2D symbologies PDF417, Data Matrix and Aztec. Where your browser provides a native barcode detector the tool uses it for speed and to find multiple codes in a single image; otherwise it falls back to well-tested open decoders, and finally to a dedicated QR reader that squeezes results out of low-contrast or awkward pictures. You do not need to tell it which type you have — it tries them for you and reports the format it found.

Reading the code is only half the job; the tool also understands the content. A decoded string is analysed and presented as structured, actionable information. A link is recognised as a URL and shown with an Open button and a validity check, so you can see the exact destination before you visit it — an important defence against malicious QR codes. A WiFi payload is broken out into network name, security type and password; a vCard becomes a tidy contact card with name, phone, email and organisation; a calendar event shows its title, time and location; and email, phone, SMS and geo-location payloads each get the right action. Anything else is shown as plain text you can copy. Every result offers copy, open, share (using your device's native share sheet where available) and print, so the decoded value goes wherever you need it.

The workspace is designed for real images, not just perfect ones. A highlight overlay draws the outline of the detected code on the preview so you can confirm the tool matched the right thing, which is especially useful when a photo contains several codes or a lot of background. Zoom, rotate and full-screen controls let you inspect and straighten a code, and a rescan button re-runs detection after you adjust the view. You can drop several images at once for batch decoding, and every result is collected into a list you can review and export together. A scan history keeps your recent results on the device so you can come back to them, and you can export or clear that history whenever you like.

Export is built in. A single result, a batch, or your whole history can be saved as TXT (just the values), CSV (value, format, type and time — ideal for a spreadsheet or inventory) or JSON (structured data for developers and automation). Because the tool detects the content type, the exported CSV and JSON include that classification, so a stock-take of barcodes or a log of scanned links is ready to use immediately. Everything is produced on the fly in the browser; no export ever touches a server.

Privacy is fundamental. Unlike many online scanners that upload your image to a server to decode it — or camera-based apps that stream video to the cloud — this tool does everything locally. Your image is read into the browser, decoded on a canvas, and the result handed straight back to you; camera frames are processed in memory and never recorded or transmitted. There is no account, nothing about your scans is tracked, and the scan history lives only in your own browser storage, which you control. That makes it safe for sensitive codes: private WiFi passwords, personal contacts, payment links and confidential documents. It keeps working offline once loaded, and the interface follows modern accessibility standards with keyboard-operable controls, ARIA labelling, visible focus outlines and high-contrast support. Note that this is a genuine, deterministic decoding tool — there are no AI features; it reads the actual data encoded in the symbol, so results are exact and repeatable. In short, it is a fast, capable and genuinely private QR and barcode scanner that lives entirely in your browser.

🛠️ Troubleshooting

  • No code found: use a clearer, larger or higher-contrast image; crop close to the code and rescan.
  • Barcode not read: some 1D/2D types depend on the browser — try Chrome or Edge, or a sharper image.
  • Camera won't start: allow camera permission, and note that camera access needs a secure (https) page.
  • Blurry photo: hold steady, tap to focus, or take the photo again with more light.
  • Tilted or rotated code: use the Rotate button and press Rescan, or straighten it first.
  • Wrong code picked in a busy image: crop to a single code, or check the "all codes found" list.
  • Link looks suspicious: don't open it — the tool shows the full URL precisely so you can judge safely.
