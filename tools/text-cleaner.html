ℹ️ About This Tool

Text copied from a PDF, a website, or an old email often comes with messy formatting baked in — random blank lines scattered throughout, double or triple spaces where there should be one, trailing spaces at the end of lines that are invisible but still there, or tab characters mixed in unpredictably with regular spaces. None of this is usually visible at a glance, but it causes real problems later: messy spacing in a document, broken alignment in code, or odd-looking gaps when the text is pasted somewhere else.

This tool cleans all of that up in a single pass, with each type of cleanup as an optional checkbox so you control exactly what gets changed. Removing blank lines collapses any sequence of empty lines down to nothing, trimming spaces clears invisible whitespace at the start and end of every line, collapsing multiple spaces turns any run of spaces into a single one, and the tab-to-space option normalizes tab characters that might be causing inconsistent indentation.

How to use:

  1. Paste your text into the input box.
  2. Choose which cleanup options you want (all are on by default except tab conversion).
  3. Click "Clean Text," then copy the result or download it as a .txt file.

Common situations this helps with

Pasting text copied from a PDF into a Word document or an email often leaves behind odd blank lines after every sentence because the PDF's internal line breaks get carried over — this tool clears that up instantly. It's also handy for cleaning up code or data that was copy-pasted from a different source with inconsistent spacing, or for tidying up a long block of notes before sharing it somewhere that should look clean and professional.

Because every cleanup option is its own checkbox, you can be as conservative or as thorough as you need — someone preparing code samples might want only tab-to-space conversion without touching blank lines that are meant to separate logical sections, while someone preparing prose for a document might want every option turned on to get the tightest possible formatting. The cleaning happens instantly on the text you've pasted, so you can experiment with different combinations of options and immediately see the result before copying or downloading the final version, with nothing ever leaving your browser in the process, which also makes it safe to use on confidential drafts or unpublished material.
