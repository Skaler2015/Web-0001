ℹ️ About This Tool

Turn a sequence of photos into a looping animated GIF — great for quick before/after sequences, simple animations, or photo-burst highlights. Reorder frames and set how long each one displays.

How to use:

  1. Select multiple images, in the order you want them to play.
  2. Reorder frames if needed, and set the delay between frames.
  3. Click Generate GIF, then download the result.

इस्तेमाल कैसे करें:

  1. जिस क्रम में चलानी हैं उस क्रम में कई इमेज चुनें।
