ℹ️ About This Tool

About the PDF Info Tool (English)

Every PDF carries a great deal of hidden information beyond the words and pictures you see. There is metadata that names the author and the software that made it, structural data that records how many pages there are and how big each one is, a security layer that can restrict printing or copying, lists of the fonts and images the file relies on, and much more. The ApneSoftware PDF Info tool surfaces all of it in one place, turning a mysterious file into a clear, readable report — and it does so entirely inside your browser, without uploading your document anywhere.

At the top of the report you get quick summary cards for the numbers you most often need — the page count, the file size, the PDF version and whether the file is encrypted — so you can size up a document at a glance. Below that, a set of grouped, searchable tables lays out the full picture, organised into logical sections so you can find exactly the detail you want.

The file and document section covers the basics and the identity of the file: its name, size, MIME type and extension, the exact PDF specification version it conforms to, and the standard document-information metadata — title, author, subject, keywords, creator and producer, along with the creation and modification dates. The “creator” usually names the program the content was authored in, while the “producer” names the library or engine that wrote the actual PDF, which is invaluable when you are troubleshooting where a file came from or why it behaves oddly.

The page analysis looks at the physical structure. It reports the total number of pages, the size of the pages in points (and the common paper name where it matches A4, Letter and so on), the orientation, any rotation applied, and whether the document has mixed page sizes — a common surprise in files assembled from different sources. Because it measures pages individually, it can tell you when a document is not as uniform as it looks.

Two sections dig into the resources the file uses. The fonts section lists the typefaces the document draws with, together with their type (such as Type 1, TrueType or CID/Type 0) and whether each appears to be embedded or subset into the file — important for knowing whether the PDF will render identically on another machine. The images section counts the embedded raster images the pages reference, so you can tell a text-only document from an image-heavy one at a glance (for pixel-level image details, our Extract Images tool goes further).

The security section is one of the most useful. It tells you whether the PDF is encrypted, and — where the file declares them — the specific permissions the owner set: whether printing, copying text, editing content and adding annotations are allowed or restricted. It also flags whether the document contains an interactive form or a digital signature field. This lets you understand at a glance why you might be unable to copy text from, or print, a particular file.

Rounding out the report, a structure section counts the document's outline (bookmarks / table of contents) entries and any file attachments, and detects standards conformance such as PDF/A (the archival standard) from the file's XMP metadata where present. A content section reads the text layer to give you the word and character counts, the average words per page and an estimated reading time — handy for writers, editors and anyone gauging the length of a document.

Getting the information out is easy. A search box filters the tables as you type, so you can jump straight to a single field in a long report. Every value has a one-click copy button, and toolbar buttons let you copy the whole report or export it as JSON (for developers and automation), CSV (for spreadsheets) or plain TXT, as well as print it. The exported report is a clean, structured record you can keep, share or attach to a ticket.

As with all ApneSoftware tools, privacy is built in. Your PDF is read and analysed entirely on your own device using JavaScript; nothing is uploaded, stored or logged, so the tool is safe for contracts, medical records, financial statements and any confidential file. Password-protected PDFs that you are authorised to open are handled locally, larger documents are read with a progress indicator, and there is no account to create, no software to install and no watermark added. Whether you are a developer inspecting output, a professional auditing documents, or simply curious about a file someone sent you, PDF Info gives you the complete story in seconds.

❓ Frequently Asked Questions

🔒 Privacy

This tool is 100% client-side. Your PDF is read and analysed entirely in your browser using JavaScript — it is never sent to ApneSoftware or any third party. Nothing is stored, logged or transmitted. When you close or refresh the page, the file is cleared from memory. Because there is no upload, you can safely inspect confidential, legal, medical and financial documents.

🌐 Browser Compatibility

Works in all modern browsers including Google Chrome, Microsoft Edge, Mozilla Firefox, Safari, Brave and Opera, on Windows, macOS, Linux, Android and iOS. It relies on standard web technologies (File API and JavaScript PDF libraries), so no plugins or extensions are required. For very large PDFs a desktop browser with more available memory gives the smoothest experience.

📁 Supported Formats

Input: PDF (.pdf), including password-protected PDFs you are authorised to open. Report export: JSON (.json), CSV (.csv) and plain text (.txt), plus copy-to-clipboard and print. The tool only reads the file; it never modifies your PDF.
