2. About This Tool | इस टूल के बारे में

यह टूल IFSC (Indian Financial System Code) की संरचना को सत्यापित करता है और उसके Prefix से बैंक का नाम पहचानता है, बैंक के नाम से उसका IFSC Prefix खोजने में मदद करता है, MICR और SWIFT/BIC कोड के फॉर्मेट की जाँच करता है, और चेक/पासबुक की तस्वीर से IFSC कोड स्कैन कर सकता है। | This tool validates the structure of an IFSC (Indian Financial System Code) and identifies the bank name from its prefix, helps you find a bank's IFSC prefix by searching its name, checks the format of MICR and SWIFT/BIC codes, and can scan a cheque/passbook image for an IFSC code.

IFSC की संरचना | Structure of an IFSC Code

हर IFSC Code 11 अक्षरों का होता है — पहले 4 अक्षर बैंक की पहचान करते हैं, 5वाँ अक्षर हमेशा "0" होता है (RBI द्वारा भविष्य के उपयोग के लिए आरक्षित), और अंतिम 6 अक्षर शाखा की पहचान करते हैं। | Every IFSC Code is 11 characters long — the first 4 characters identify the bank, the 5th character is always "0" (reserved by RBI for future use), and the last 6 characters identify the branch.

बैंक Prefix और शाखा Code | Bank Prefix and Branch Code

यह टूल 55 से अधिक प्रमुख भारतीय बैंकों की एक सत्यापित सूची का उपयोग करके Prefix से बैंक का नाम दिखाता है — यह किसी विशेष शाखा का पता, संपर्क विवरण, MICR या SWIFT/BIC कोड प्रदान नहीं करता, क्योंकि इस टूल में कोई सत्यापित शाखा-स्तरीय डेटाबेस शामिल नहीं है। | This tool uses a verified list of 55+ major Indian banks to show the bank name from the prefix — it does not provide a specific branch's address, contact details, MICR or SWIFT/BIC code, since no verified branch-level database is bundled in this tool.

यदि आपका IFSC Code संरचनात्मक रूप से सही है लेकिन उसका Prefix हमारी सूची में नहीं है, तो यह टूल इसे ईमानदारी से "unverified" के रूप में दिखाता है, बजाय किसी गलत बैंक नाम का अनुमान लगाने के। | If your IFSC Code is structurally valid but its prefix isn't in our list, this tool honestly shows it as "unverified" rather than guessing at an incorrect bank name.

IFSC बनाम MICR बनाम SWIFT | IFSC vs MICR vs SWIFT

IFSC (11 अक्षर) घरेलू इलेक्ट्रॉनिक ट्रांसफर (NEFT/RTGS/IMPS) के लिए शाखा की पहचान करता है। MICR (9 अंक) चेक की भौतिक प्रोसेसिंग के लिए है। SWIFT/BIC (8 या 11 अक्षर) अंतरराष्ट्रीय वायर ट्रांसफर के लिए बैंक की पहचान करता है। तीनों अलग-अलग सिस्टम हैं और एक से दूसरे की गणना नहीं की जा सकती। | IFSC (11 characters) identifies the branch for domestic electronic transfers (NEFT/RTGS/IMPS). MICR (9 digits) is for physical cheque processing. SWIFT/BIC (8 or 11 characters) identifies the bank for international wire transfers. All three are separate systems and cannot be calculated from one another.

NEFT, RTGS, IMPS और UPI में अंतर | Difference between NEFT, RTGS, IMPS and UPI

NEFT बैचों में प्रोसेस होता है और छोटी-बड़ी दोनों राशियों के लिए उपयुक्त है। RTGS बड़ी राशियों (₹2 लाख+) के लिए वास्तविक समय में प्रोसेस होता है। IMPS तुरंत प्रोसेस होता है और 24x7 उपलब्ध है। UPI को IFSC की आवश्यकता नहीं होती — यह UPI ID या मोबाइल नंबर उपयोग करता है। | NEFT is processed in batches and suits both small and large amounts. RTGS processes large amounts (₹2 lakh+) in real time. IMPS processes instantly and is available 24x7. UPI does not require an IFSC code — it uses a UPI ID or mobile number instead.

सुरक्षा सुझाव | Safety Tips

महत्वपूर्ण लेन-देन से पहले हमेशा अपने बैंक की आधिकारिक वेबसाइट, पासबुक, चेकबुक या नेटबैंकिंग पोर्टल से अपने सटीक IFSC Code की पुष्टि करें। कभी भी किसी अनजान स्रोत से मिले IFSC Code पर भरोसा न करें। | Always confirm your exact IFSC Code from your bank's official website, passbook, chequebook or net-banking portal before an important transaction. Never trust an IFSC code received from an unknown or unverified source.

सामान्य गलतियाँ | Common Errors

सबसे आम गलतियाँ हैं: 5वें अक्षर की जगह "0" के बजाय "O" (letter) टाइप करना, पुराने विलय हुए बैंक का Prefix इस्तेमाल करना, या शाखा कोड में एक अक्षर छूट जाना। | The most common mistakes are: typing the letter "O" instead of the digit "0" as the 5th character, using an old merged bank's prefix, or missing a character in the branch code.

सर्वोत्तम प्रथाएँ | Best Practices

ट्रांसफर से पहले हमेशा प्राप्तकर्ता के खाता संख्या और नाम के साथ IFSC को दोबारा जाँचें, और यदि संभव हो तो पहले एक छोटी राशि भेज कर परीक्षण करें। | Always double-check the IFSC alongside the recipient's account number and name before a transfer, and if possible, test with a small amount first.
