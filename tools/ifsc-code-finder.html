2. About This Tool | इस टूल के बारे में

यह टूल IFSC (Indian Financial System Code) की संरचना को सत्यापित करता है और उसके Prefix से बैंक का नाम पहचानता है, या बैंक के नाम से उसका IFSC Prefix खोजने में मदद करता है। | This tool validates the structure of an IFSC (Indian Financial System Code) and identifies the bank name from its prefix, or helps you find a bank's IFSC prefix by searching its name.

हर IFSC Code 11 अक्षरों का होता है — पहले 4 अक्षर बैंक की पहचान करते हैं, 5वाँ अक्षर हमेशा "0" होता है (RBI द्वारा भविष्य के उपयोग के लिए आरक्षित), और अंतिम 6 अक्षर शाखा की पहचान करते हैं। | Every IFSC Code is 11 characters long — the first 4 characters identify the bank, the 5th character is always "0" (reserved by RBI for future use), and the last 6 characters identify the branch.

यह टूल 55 से अधिक प्रमुख भारतीय बैंकों की एक सत्यापित सूची का उपयोग करके Prefix से बैंक का नाम दिखाता है — यह किसी विशेष शाखा का पता या संपर्क विवरण प्रदान नहीं करता, क्योंकि इस Tool में कोई सत्यापित शाखा-स्तरीय डेटाबेस शामिल नहीं है। | This tool uses a verified list of 55+ major Indian banks to show the bank name from the prefix — it does not provide a specific branch's address or contact details, since no verified branch-level database is bundled in this tool.

यदि आपका IFSC Code संरचनात्मक रूप से सही है लेकिन उसका Prefix हमारी सूची में नहीं है, तो यह टूल इसे ईमानदारी से "unverified" के रूप में दिखाता है, बजाय किसी गलत बैंक नाम का अनुमान लगाने के। | If your IFSC Code is structurally valid but its prefix isn't in our list, this tool honestly shows it as "unverified" rather than guessing at an incorrect bank name.

NEFT, RTGS और IMPS जैसे इलेक्ट्रॉनिक फंड ट्रांसफर के लिए सही IFSC Code का होना ज़रूरी है, ताकि पैसा सही बैंक शाखा तक पहुँच सके। | Having the correct IFSC Code is essential for electronic fund transfers like NEFT, RTGS and IMPS, so that money reaches the correct bank branch.

महत्वपूर्ण लेन-देन से पहले हमेशा अपने बैंक की आधिकारिक वेबसाइट, पासबुक, चेकबुक या नेटबैंकिंग पोर्टल से अपने सटीक IFSC Code की पुष्टि करें। | Always confirm your exact IFSC Code from your bank's official website, passbook, chequebook or net-banking portal before an important transaction.

हर गणना पूरी तरह आपके ब्राउज़र के भीतर होती है — कोई भी IFSC Code या Search Query कभी किसी सर्वर पर अपलोड नहीं की जाती, और यह टूल किसी भी AI-आधारित सुविधा का उपयोग नहीं करता। | Every calculation runs entirely within your browser — no IFSC Code or search query is ever uploaded to a server, and this tool does not use any AI-powered features.

3. Frequently Asked Questions | अक्सर पूछे जाने वाले प्रश्न

Q. What is an IFSC Code? | IFSC Code क्या है?
An 11-character code identifying a specific bank branch for electronic fund transfers in India. | भारत में इलेक्ट्रॉनिक फंड ट्रांसफर के लिए किसी विशेष बैंक शाखा की पहचान करने वाला 11-अक्षर का कोड।

Q. What does IFSC stand for? | IFSC का पूरा नाम क्या है?
Indian Financial System Code. | Indian Financial System Code।

Q. What is the structure of an IFSC Code? | IFSC Code की संरचना क्या है?
4 letters for the bank, followed by "0", followed by 6 characters for the branch. | बैंक के लिए 4 अक्षर, उसके बाद "0", उसके बाद शाखा के लिए 6 अक्षर।

Q. Why is the 5th character always 0? | 5वाँ अक्षर हमेशा 0 क्यों होता है?
It is reserved by the RBI for potential future use. | यह RBI द्वारा भविष्य के संभावित उपयोग के लिए आरक्षित है।

Q. Where can I find my bank's IFSC Code? | मैं अपने बैंक का IFSC Code कहाँ पा सकता हूँ?
On your chequebook, passbook, or your bank's net-banking portal. | अपनी चेकबुक, पासबुक, या अपने बैंक के net-banking पोर्टल पर।

Q. Is the IFSC Code the same for all branches of a bank? | क्या किसी बैंक की सभी शाखाओं का IFSC Code समान होता है?
No — only the first 4 letters (bank code) are common; the last 6 characters differ by branch. | नहीं — केवल पहले 4 अक्षर (bank code) समान होते हैं; अंतिम 6 अक्षर शाखा के अनुसार भिन्न होते हैं।

Q. Does this tool show the branch address? | क्या यह टूल शाखा का पता दिखाता है?
No — it only decodes the bank name from the prefix; branch-level details require checking with your bank. | नहीं — यह केवल Prefix से बैंक का नाम दिखाता है; शाखा-स्तरीय विवरण के लिए अपने बैंक से जाँच आवश्यक है।

Q. What happens if the bank prefix isn't recognized? | अगर बैंक Prefix पहचाना न जाए तो क्या होता है?
The tool honestly flags it as unverified rather than guessing — it may still be valid for a smaller or regional bank. | Tool इसे अनुमान लगाने के बजाय ईमानदारी से unverified के रूप में flag करता है — यह किसी छोटे या क्षेत्रीय बैंक के लिए अभी भी मान्य हो सकता है।

Q. Is IFSC Code required for NEFT? | क्या NEFT के लिए IFSC Code आवश्यक है?
Yes — it identifies the exact receiving branch for the transfer. | हाँ — यह ट्रांसफर के लिए सटीक प्राप्तकर्ता शाखा की पहचान करता है।

Q. Is IFSC Code required for RTGS? | क्या RTGS के लिए IFSC Code आवश्यक है?
Yes, just like NEFT and IMPS. | हाँ, NEFT और IMPS की तरह ही।

Q. Is IFSC Code required for IMPS? | क्या IMPS के लिए IFSC Code आवश्यक है?
Yes, when transferring using account number and IFSC (rather than mobile/MMID or UPI). | हाँ, जब account number और IFSC का उपयोग करके ट्रांसफर किया जाए (mobile/MMID या UPI के बजाय)।

Q. Can IFSC codes change over time? | क्या IFSC codes समय के साथ बदल सकते हैं?
Yes — bank mergers and branch restructuring can change IFSC codes; always verify the current one. | हाँ — बैंक विलय और शाखा पुनर्गठन से IFSC codes बदल सकते हैं; हमेशा वर्तमान कोड की पुष्टि करें।

Q. What happened to banks that merged, like Vijaya Bank? | Vijaya Bank जैसे विलय हुए बैंकों का क्या हुआ?
Their old IFSC prefixes may still appear on older cheques, but new accounts use the merged bank's codes. | उनके पुराने IFSC prefixes पुराने cheques पर अभी भी दिख सकते हैं, लेकिन नए accounts विलय की गई बैंक के codes उपयोग करते हैं।
