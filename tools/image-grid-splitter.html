ℹ️ About This Tool

Splitting a single image into a grid of equal pieces has a few popular uses: the most common is the Instagram "grid post" trend, where a single wide photo or piece of artwork is cut into a 3×3 grid so that posting all nine pieces in order recreates the full image across your profile feed. It's also useful for printing a large image like a poster or banner across multiple sheets of paper that get taped together afterward, or simply for breaking a wide screenshot or photo into smaller chunks to share individually where only small images are accepted.

Upload a photo, choose how many rows and columns to split it into — either pick a common preset like the 3×3 Instagram grid, or set a custom grid for any other layout — and the grid lines overlay the live preview so you can see exactly where each cut will happen before splitting. Clicking Split & Download gives you every piece bundled together as a single ZIP file, numbered in reading order (left to right, top to bottom), so you always know which piece goes where when reassembling or posting them in sequence.

How to use:

  1. Upload a photo.
  2. Pick a preset grid size or enter custom rows and columns.
  3. Click Split & Download ZIP to get every piece as separate numbered image files.

Tips for a clean Instagram grid

For the classic 3×3 Instagram layout, start with a perfectly square source image if possible — Instagram crops posts to a square by default, so a non-square source can throw off the alignment between pieces once posted. Upload the resulting nine pieces to Instagram in reverse order (bottom-right piece first, ending with the top-left piece last), since the most recent post appears at the top-left of a profile grid — posting in reverse is what makes the final feed line up correctly into the original image.

The custom rows and columns option isn't limited to square grids either — a 1×3 row works well for a wide panorama split into a horizontal triptych, while a 3×1 column suits a tall image split for a vertical carousel post. Whatever the layout, the live grid overlay on the preview always shows exactly where each cut will land before you commit, so there are no surprises in the downloaded pieces once the ZIP file is unzipped and reviewed on your computer or phone.

साफ़ Instagram grid के लिए सुझाव

Classic 3×3 Instagram layout के लिए, अगर मुमकिन हो तो एक बिल्कुल square source image से शुरू करें — Instagram posts को default रूप से square में crop करता है, तो non-square source post होने के बाद हिस्सों के alignment को बिगाड़ सकता है। बनी हुई नौ pieces को Instagram पर उल्टे क्रम में upload करें (bottom-right वाला हिस्सा पहले, top-left वाला आख़िर में), क्योंकि सबसे नया post profile grid में सबसे ऊपर-बाएं दिखता है — उल्टे क्रम में post करने से ही आख़िर में feed सही जुड़कर original image बनती है।

Custom rows-columns option सिर्फ़ square grids तक सीमित नहीं है — एक 1×3 row किसी चौड़ी panorama image को horizontal triptych में बांटने के लिए अच्छी है, जबकि 3×1 column किसी लंबी image को vertical carousel post के लिए उपयुक्त है। कोई भी layout हो, preview पर live grid overlay हमेशा दिखा देता है कि हर कट commit करने से पहले कहां पड़ेगा, तो downloaded pieces में कोई हैरानी नहीं होती।
