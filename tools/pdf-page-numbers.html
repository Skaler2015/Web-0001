ℹ️ About This Tool

About the PDF Page Numbers Tool (English)

Page numbers seem like a small detail, but they do a lot of quiet work. They let a reader cite “see page 14”, keep a printed stack of paper in the right order after it is dropped, tell a reviewer exactly where a change belongs, and give a report, contract, dissertation or ebook the finished, professional feel that an unnumbered document lacks. Adding them by hand — editing each page in a word processor and re-exporting — is slow, and it is impossible when all you have is the finished PDF. The ApneSoftware PDF Page Numbers tool solves that: it stamps clean, consistent numbers onto every page you choose, in the format and position you want, entirely inside your web browser.

The tool supports a wide range of number formats so it fits whatever convention your document follows. Use a plain 1, 2, 3; zero-padded 01 or 001 for a tidy fixed-width look; the descriptive Page 1, Page 01 or Page 001; the reference-friendly 1 of N or 1 / N that shows the total; classic Roman numerals in lower or upper case (i, ii, iii or I, II, III) for prefaces and front matter; or letters (a, b, c or A, B, C) for appendices and annexes. On top of any format you can add a custom prefix and suffix — for example a dash on each side to produce “- 5 -”, or the word “Sheet ” before the number — so the label reads exactly the way your organisation expects.

Where the number sits is completely up to you. A simple nine-point grid covers the header and footer at the left, centre and right, and the three middle positions if you ever need a side marker. A margin slider moves the number in from the edge of the page, and if you need pixel-precise placement you can switch on custom X and Y coordinates (measured in PDF points from the bottom-left corner) to drop the number anywhere. The styling controls then let you match the look of your document: choose a sans-serif, serif or monospace font, make it bold, italic or underlined, adjust the size and letter spacing, set the opacity, and pick any colour with a visual picker or an exact hex code. A subtle grey footer number and a bold black header number are both a couple of clicks away.

Real documents rarely want a number on literally every page, so the tool includes flexible page targeting. Number all pages; skip the first page when it is a cover or title page (numbering then begins on the second page); number only the first or only the last page; number just the odd or just the even pages, which is useful for double-sided layouts; or type a precise custom range such as 1-3, 5, 8-10. The starting number is yours to set, and a one-click “restart at 1” option is handy when you are numbering a chapter or appendix that should begin its own sequence. When you choose a “1 of N” format, the total N reflects how many pages are actually being counted, so it stays meaningful even when you skip the cover.

A live preview renders the real page with the number drawn exactly where and how it will appear in the final file. You can page through the document and zoom in to check alignment and size, and because the number shown is the true value for that page, there are no surprises after processing. When you are ready, the output options let you rename the downloaded file, choose whether to preserve or remove the document metadata (title, author and similar fields), and optionally compress the result using object streams to keep the file size down. You can download the numbered PDF immediately or send it straight to your printer.

Because everything happens on your own device, the tool is safe for confidential material. Password-protected PDFs that you are authorised to open are handled locally, and large multi-hundred-page documents are processed page by page to keep memory use reasonable. Nothing is uploaded, nothing is stored, and no account or installation is required. The only change made to your PDF is the page numbers you asked for; every other byte of the original content, text and layout is preserved exactly as it was.

Typical users include students and academics numbering theses and research papers, businesses preparing reports, proposals and contracts, publishers and self-publishers numbering ebooks and manuscripts, and offices that need consistent numbering on scanned or merged documents before printing or archiving. Whether you need a single grey number in the footer or a formal “Page 1 of 250” across a long legal bundle, the tool produces a clean, correctly numbered PDF in seconds.

🔒 Privacy

This tool is 100% client-side. Your PDF is read and processed entirely in your browser using JavaScript — it is never sent to ApneSoftware or any third party. Nothing is stored, logged or transmitted. When you close or refresh the page, the file is cleared from memory. Because there is no upload, you can safely number confidential, legal, medical and financial documents.

🌐 Browser Compatibility

Works in all modern browsers including Google Chrome, Microsoft Edge, Mozilla Firefox, Safari, Brave and Opera, on Windows, macOS, Linux, Android and iOS. It relies on standard web technologies (Canvas, File API and JavaScript PDF libraries), so no plugins or extensions are required. For very large PDFs a desktop browser with more available memory gives the smoothest experience.

📁 Supported Formats

Input: PDF (.pdf), including password-protected PDFs you are authorised to open. Output: a standard PDF (.pdf) with your page numbers applied, ready to download or print. Number styles: 1, 01, 001, Page 1/01/001, 1 of N, 1 / N, Roman (i/I) and letters (a/A), with optional prefix and suffix.
