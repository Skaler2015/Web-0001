ℹ️ About This Tool

The Image Color Palette Extractor is a fast, private, browser-based colour tool that turns any photo into a usable, well-documented palette — and then helps you actually work with those colours. Designers, developers, marketers and photographers constantly need to pull colours out of an image: to match a brand from a logo, to build a website theme from a hero photo, to sample a mood-board, to reproduce a print colour, or simply to find a set of colours that already work together because they came from the same picture. Doing this by eye is slow and imprecise. This tool analyses the actual pixels with the browser's Canvas API, quantises them into their most representative colours, and presents a clean palette with every colour value you could need — instantly, and without ever uploading your image anywhere.

Extraction is flexible. You choose how many colours you want — from a quick set of three up to thirty or more, via handy size presets or a custom count — and pick an extraction mode: Dominant returns the most frequent colours, Vibrant favours the most saturated and lively ones, Muted pulls the soft, desaturated tones, and Light and Dark bias toward the bright or shadow end of the image. The engine avoids near-duplicate swatches so the palette stays varied, and it computes the image's average colour too. Alongside the swatches, an Image Analysis panel reports the overall brightness, the average colour, the dominant colour's share of the image, and how many distinct colour regions were found — useful context when you are judging whether a palette truly represents a picture.

Every colour is fully described. Click a swatch and the Colour Details panel shows it in HEX, RGB, RGBA, HSL, HSV, CMYK, LAB and XYZ — each one click-to-copy — along with the nearest colour name and its brightness, saturation, hue and relative luminance. Because accessible design depends on contrast, a WCAG panel gives the contrast ratio of the colour against black and white text and marks whether it passes AA and AAA, with a clear recommendation of which text colour to use on top of it. You can copy the colour as a CSS custom property or as the nearest Tailwind-style value, and mark it as a favourite.

The suite goes beyond extraction into real colour work. A harmony generator builds complementary, analogous, triadic, split-complementary, tetradic, square and monochromatic schemes plus shades, tints and tones around any selected colour — perfect for expanding a single brand colour into a full system. A gradient generator turns the whole palette into a ready-to-use linear, radial or conic CSS gradient. A brand-roles panel suggests a primary, secondary, accent, neutral and call-to-action colour from the palette, giving you a head start on a design system. And a live search lets you filter the palette by hex code or colour name.

When it is time to hand colours off, the export options cover every workflow. Copy the palette as a plain HEX list, as CSS custom properties, as SCSS variables, as a Tailwind config snippet, or as JSON. Download it as a PNG or SVG palette image, as CSS/SCSS/LESS, as TXT or JSON, as a GIMP GPL palette (which many editors import), or as an Adobe ASE swatch file for Photoshop, Illustrator and other Adobe apps. You can print a colour sheet, and a local history keeps your recent palettes and favourites in your browser for reuse. Everything runs as plain JavaScript on your own device, so it is lightweight and instant, works offline once loaded, and is completely private — your images and colours are never sent to a server, there is no sign-up, and nothing is tracked.

Who is it for? UI and web designers building themes and design systems; graphic and brand designers matching and documenting colours; front-end developers who need exact CSS values and accessible contrast; digital marketers creating on-brand graphics; print professionals who need CMYK and LAB; photographers analysing the palette of a shot; and students learning colour theory. Whether you need one hex code in five seconds or a fully documented, exportable, accessible palette, this tool turns any image into a colour system you can trust and reuse. It is deliberately lightweight — no heavy frameworks, no external colour service — so it opens fast, responds instantly as you change modes or counts, and behaves the same on a phone, a tablet or a wide desktop monitor. Because the extraction, the format conversions, the harmony maths and almost every export all run locally, you can rely on it for confidential brand work, unreleased product photos or client images without any of them leaving your machine, and you can keep using it on a plane or anywhere else without a connection.

❓ Frequently Asked Questions

🛠️ Troubleshooting

  • Palette looks washed out: Try the Vibrant mode, or increase the colour count so distinct tones are not merged.
  • Too many similar colours: Lower the count or switch modes; the tool already removes near-duplicates, but very uniform photos yield close colours.
  • Colours differ slightly from the image: Extraction quantises pixels for speed, so swatches are representative rather than exact single pixels — use a colour picker for one exact pixel.
  • CMYK looks off in print: On-screen CMYK is an approximation; always proof with your printer's colour profile.
  • ASE won't import: Use the GPL or JSON export as a fallback, or the PNG palette which any tool can sample.
  • Large image is slow: The image is downscaled before analysis, but very large files still take a moment on older devices; resize first if needed.
  • Saved palettes gone: They live only in your browser; clearing site data or private mode will not retain them.
