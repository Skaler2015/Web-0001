🗺️ What is an XML Sitemap?

An XML sitemap is a file that lists all the important pages on your website, helping search engines discover and crawl them faster and more efficiently. Without a sitemap, search engines must discover your pages entirely through links, which can miss pages that aren't well linked.

A sitemap is especially important for: large websites (100+ pages), new websites with few backlinks, websites with rich media content, websites with frequently updated content, and websites with complex navigation.

Sitemap Tags Explained

  1. loc — Full URL of the page (required)
  2. lastmod — Date the page was last modified
  3. changefreq — How often the page changes
  4. priority — Relative importance (0.0–1.0)

After Generating

  1. Upload sitemap.xml to your website root
  2. Add Sitemap directive to your robots.txt
  3. Submit to Google Search Console
  4. Submit to Bing Webmaster Tools

Priority कैसे set करें?

  1. Homepage: 1.0
  2. Category pages: 0.8
  3. Product/Blog pages: 0.6-0.7
  4. Tag/Archive pages: 0.4-0.5
  5. Legal pages: 0.3

Submit कैसे करें?

  1. sitemap.xml को website root पर upload करें
  2. robots.txt में Sitemap URL add करें
