The Professional PDF Form Flattening Tool takes a filled-in PDF form and makes its values permanent. "Flattening" converts interactive fields — text boxes, checkboxes, radio buttons, dropdowns and more — into fixed page content, so the answers can no longer be edited, cleared or accidentally changed after submission. Beyond the one-click flatten, it analyses every field in your document, lets you add a watermark, edit or strip metadata, optimise the file size, preview the result before and after, and process several PDFs at once — all inside your browser, with the file never leaving your device.

⚙️ How it works

When you add a PDF, the tool loads it with pdf-lib — a pure-JavaScript PDF library — and inspects its AcroForm to list every field, its type and its current value. When you flatten, pdf-lib first refreshes each field's visual appearance so what you see is exactly what gets baked in, then renders those appearances into the page content and removes the interactive widgets. The result is a normal PDF whose former field values are now ordinary, permanent page text and graphics. If you choose read-only lock instead, the fields are kept but marked non-editable. Watermarks and metadata changes are applied to the same document before it is saved, and a before/after preview is rendered with PDF.js so you can confirm the outcome.

📄 Supported PDFs

Any standard PDF with an AcroForm — the fillable forms produced by Adobe Acrobat, LibreOffice, government and bank portals, HR and tax software and most form designers — can be flattened. The tool detects and reports text fields, checkboxes, radio-button groups, dropdowns, list boxes, push buttons and signature fields. PDFs that use XFA (a legacy Adobe-only form technology) may not expose their fields to standard tooling; if a form does not flatten, re-save it as a standard PDF first. Encrypted (password-protected) PDFs must be unlocked before flattening.

Because everything runs locally, none of your sensitive form data is exposed during the process.

ℹ️ About This Tool

The Professional PDF Form Flattening Tool is a free, browser-based utility that turns a filled-in PDF form into a locked, permanent document. Interactive PDF forms are wonderfully convenient while you are completing them, but that same interactivity is a liability once the form is done: any recipient can reopen it and change a figure, clear a checkbox or overwrite a name, sometimes without leaving a trace. Flattening solves this by converting the form's live fields into ordinary page content, so the answers you submitted are exactly the answers everyone sees, forever. This tool began as a one-click flattener and has grown into a complete form-processing workspace, adding field analysis, validation, watermarking, metadata control, optimisation, a before/after preview and batch processing — while keeping the whole operation on your own device.

The first thing the tool does when you add a PDF is understand it. Using pdf-lib, an open-source PDF engine written entirely in JavaScript, it reads the document's form definition and produces a clear inventory: how many fields there are, and how many of each type — text fields, checkboxes, radio-button groups, dropdowns, list boxes, push buttons and signature fields. It shows this both as a set of type chips and as a scrollable table listing each field's name, type and current value, flagging which fields are filled, which are empty and which are marked required. This matters because flattening is irreversible; seeing exactly what is in the form, and whether anything required was left blank, lets you catch mistakes before you lock everything in. A PDF-information panel sits alongside it with the file name, size, PDF version, page count, total field count and the number of signature fields.

Flattening itself offers two modes to suit different needs. The default, permanent flatten, is the classic behaviour: pdf-lib refreshes every field's appearance so the rendered values match what you see, then draws those appearances into the page and strips out the interactive form, producing a standard PDF whose former answers are now fixed text and graphics. Because the values are rendered as real text, the flattened document remains searchable and selectable, which is important for accessibility and for anyone who needs to copy information out later. The alternative, read-only lock, keeps the form fields in place but marks them non-editable — a lighter touch for cases where you want to discourage changes but preserve the form's structure. A dedicated option lets you preserve signature fields, skipping them so an existing signature widget is not disturbed by the flatten.

Around the core flatten sit the finishing touches that a professional workflow needs. A watermark can be stamped across every page — ready-made Confidential, Approved and Submitted labels, or any custom text — at an opacity you control, which is invaluable for marking the status of applications, contracts and internal documents. Metadata handling gives you three choices: keep the document's existing title, author and subject; remove them entirely to sanitise a file before sharing; or edit them to set your own. An optimisation option re-saves the PDF using object streams to reduce its size, and an OCR-safety toggle keeps the flattened text searchable so the document stays useful to screen readers and search. Throughout the process, a progress bar reports how many fields have been flattened, with an estimate of the time remaining and the processing speed for larger forms.

Confidence comes from being able to see the result, so the tool renders a first-page preview of the original document and, after flattening, of the finished file using PDF.js, side by side. When you are satisfied you download the locked PDF with a click. For anyone handling volume — an HR team processing a batch of onboarding forms, an accounts department locking a month of invoices, a school flattening exam entry forms — multiple PDFs can be added to a queue and processed together, then downloaded as a single ZIP. And because the entire pipeline is client-side, none of this touches a server: the tool is honest about its boundaries too, noting that genuine password encryption is handled by the companion PDF Protect tool rather than pretending to encrypt, and that XFA or password-protected forms need a preparatory step first.

Everything is built to be lightweight, fast and dependable on real-world files. Pages and fields are processed with memory in mind so that large, many-field forms do not overwhelm the browser; the layout is fully responsive and touch-friendly for uploading and flattening from a phone; and the controls are labelled and keyboard-operable for accessibility. Above all, it respects the sensitive nature of forms. Applications, tax returns, medical intake sheets, loan documents and contracts routinely pass through a flattening step, and doing that in a tool that never uploads your file means the personal and financial details inside it stay entirely under your control. Whether you are an individual locking a single signed form or a business standardising a document workflow, this tool turns "make this form permanent" into a couple of clicks without ever compromising privacy.

🛠️ Troubleshooting

  • "No fillable form fields": the PDF has no AcroForm or uses XFA — re-save it as a standard PDF (print to PDF) and retry.
  • Values look blank after flattening: the field appearances were not generated — the tool refreshes them automatically, but some tools produce empty appearances; open and re-save the source in a PDF viewer first.
  • The PDF won't load: it may be password-protected — remove the password with PDF Unlock first.
  • The file is still large: choose the optimised output and consider PDF Compress for image-heavy documents.
  • A signature disappeared: enable "Preserve signature fields" to skip signature widgets.
  • The browser slows on a big form: flatten one file at a time and close other tabs to free memory.
