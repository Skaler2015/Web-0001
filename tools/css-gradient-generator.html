🌈 CSS Gradient Generator

Build a linear or radial gradient visually, and copy the CSS code.

🎨 Color Stops

ℹ️ About This Tool

Visually design a CSS gradient — linear or radial, with as many color stops as you like — and get the ready-to-use CSS code to paste into your stylesheet.

How to use:

  1. Pick Linear or Radial, and set the angle for linear gradients.
  2. Add, remove, or adjust color stops and their position.
  3. Copy the generated CSS code.

ℹ️ इस टूल के बारे में

CSS gradient को विज़ुअली डिज़ाइन करें — linear या radial, जितने चाहें उतने color stops के साथ — और सीधे अपनी स्टाइलशीट में पेस्ट करने के लिए तैयार CSS कोड पाएं।

इस्तेमाल कैसे करें:

  1. Linear या Radial चुनें, और linear के लिए angle सेट करें।
  2. color stops जोड़ें, हटाएं, या उनकी पोजीशन सेट करें।
  3. बना हुआ CSS कोड कॉपी करें।
