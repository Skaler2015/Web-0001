🔍 What is Meta Tag Analyzer?

The Meta Tag Analyzer is a free SEO tool that reads any webpage's HTML source code and extracts every meta tag — then grades each one against Google's best practices. In one click, you get a full report covering your title tag, meta description, canonical URL, Open Graph tags, Twitter Card tags, robots directives, language, charset, viewport, author, theme color, and more.

Unlike tools that require a live URL (which often get blocked by CORS or server restrictions), this analyzer works entirely in your browser using the raw HTML you paste in. Just open any webpage, press Ctrl+U (View Source), select all, paste here, and hit Analyze. You get an instant, completely private report — nothing is sent to any server.

What Does the Score Mean?

The SEO score (0–100) is calculated based on the presence and quality of critical meta tags. A score above 80 means your page's meta tags are well-optimized. 50–79 means there are gaps to fix. Below 50 means important tags are missing or incorrectly formatted.

How to Use

  1. Open any webpage in your browser and press Ctrl+U to view source.
  2. Press Ctrl+A to select all, then Ctrl+C to copy.
  3. Paste the HTML into the box above and click Analyze Tags.
  4. Review the score, SERP preview, OG preview, and each tag's status.
  5. Fix any tags marked ❌ or ⚠️ and re-analyze to confirm.
  6. Download the report as a text file to share with your team.

✅ क्या-क्या Check होता है?

  1. Title Tag — लंबाई और keyword presence
  2. Meta Description — 150-160 chars की जांच
  3. Canonical URL — duplicate content से बचाव
  4. Open Graph Tags — Social sharing के लिए
  5. Twitter Card Tags — Twitter preview
  6. Robots Directives — Index/noindex जांच
  7. Viewport — Mobile-friendly check
  8. Charset — Encoding सही है या नहीं
  9. Author, Language, Theme Color
