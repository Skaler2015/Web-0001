Report a Bug

When reporting a bug, please include:

  • The tool name and URL
  • What you were trying to do
  • What happened instead
  • Your browser and device (e.g., Chrome 125 on Android)
  • File type and approximate size (for file processing tools)

Suggest a New Tool

We regularly add new tools based on user suggestions. When suggesting, tell us:

  • What the tool should do
  • Why you need it
  • Any similar tools you've seen elsewhere

We prioritize tools that can run entirely in the browser (no server upload), are useful to Indian users, and fill a gap in our current collection.

Response Time

We aim to respond to all messages within 1–3 business days. For urgent bug reports affecting many users, we try to respond within 24 hours.

Frequently Asked Questions

Can I request a custom tool for my business?
We focus on free public utility tools. For custom software development inquiries, please email us at hello@apnesoftware.com.

Do you accept content contributions?
We are not currently accepting guest blog posts or sponsored content.

Is there an API for your tools?
Our tools are client-side only and do not currently offer a public API.
